브로드캐스트 , 멀티 캐스트 - 1 : 多 통신을 하기위한 방식 - UDP를 기반으로 구현한다...브로드캐스트(BroadCast) - 서로 연결된 아이피주소에서 데이터를 주고 받는 방식...멀티캐스트(MultiCast) - 같은 그룹에 가입된 모든 개체에게 데이터를 전송하는 방식----------------------------------------------------------------------------Server - SenderClient - Receiver----------------------------------------------------------------------------Socket Options - Socket의 기본 동작을 변경하거나 세부적인 제어를 하는 옵션...옵션값을 설정하는 함수int setsockopt( SOCKET s, int level, int optname, const char * optval, int optlen ); - SOCKET s - 옵션을 적용할 소켓 - int level  - Socket Option Level  - 옵션을 적용할 분류...  - SOL_SOCKET - 브로드캐스트를 위한 옵션  - IPPROTO_TCP - TCP를 위한 옵션  - IPPROTO_IP - 멀티캐스트를 위한 옵션 - int optname - 옵션 이름 - const char * optval  - 적용할 옵션값을 저장한 변수의 주소 - int optlen  - 옵션값을 저장한 변수의 크기옵션값을 확인하는 함수int getsockopt( SOCKET s, int level, int optname, char * optval, int * optlen ); - SOCKET s - 옵션이 적용된 소켓 - int level  - Socket Option Level - int optname - 옵션 이름 - char * optval  - 옵션값을 저장할 변수의 주소 - int * optlen  - 옵션값을 저장한 변수의 크기를 저장한 변수의 주소================================================================================================SOL_SOCKET에 대한 세부 옵션 - SO_BROADCAST     BOOL        브로드캐스트를 허용할지 말지... - SO_KEEPALIVE     BOOL        연결 상태 확인 - SO_LINGER        linger{}    closesocket 함수의 리턴 지연시간 설정.. - SO_SNDBUF        int         소켓 송/수신 버퍼의 크기를 설정하는 옵션   SO_RCVBUF - SO_SNDTIMEO      int         데이터 송/수신 함수 호출시 timeout값 설정...   SO_RCVTIMEO===========================================================================================================멀티 캐스트 - 같은 그룹에 가입된 모든 개체에게 데이터를 전송하는 방식 - 그룹 가입과 탈퇴는 클라이언트 쪽에서 해야 한다... - 그룹 구성원 모두가 평등하다... - TTL 값은 최소 2로 설정한다... - 브로드캐스트와 같이 멀티캐스트는 사용시 예약된 IP주소를 사용해야 한다...  - 224.0.0.0 ~ 239.255.255.255 범위의 예약된 IP주소 대역.... ==========================================================================================================IPPROTO_IP에 대한 세부 옵션.... - IP_MULTICAST_IF      IN_ADDR{}  멀티 캐스트 패킷을 보낼 인터페이스 선택... - IP_MULTICAST_LOOP    BOOL       멀티캐스트 루프백 여부 - IP_ADD_MEMBERSHIP    IP_MREQ{}   멀티 캐스트 그룹 가입 - IP_DROP_MEMBERSHIP   IP_MREQ{}   멀티 캐스트 그룹 탈퇴 - IP_MULTICAST_TTL     int         멀티캐스트의 TTL값 설정.===========================================================================================================typedef struct ip_mreq {    IN_ADDR imr_multiaddr;  // 멀티캐스트 주소    IN_ADDR imr_interface;  // 현재 아이피 주소} IP_MREQ, *PIP_MREQ;
